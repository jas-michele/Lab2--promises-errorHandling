## Promises & Error Handling (TypeScript)
This project demonstrates handling asynchronous operations usng Promises in TyeScript, along with proper error handling techniques.

## Type Safety
TypeScript enforces types on Promises responses and function return values, ensuring consistent and predictable async data handling.

## Promises 
Promises are used to manage asynchronus operations, allowing code to handle success (.then) and  failure (.catch) without blockinng execution.

## Error Handling 
Errors are handled using .catch() and try.. catch with async/await preventing crashes and improving application stability.

## Async/Await
async/await simplifies Promise handling by making asynchronus code more readable and easier to debug.
